axiam_error_kind_t axiam_jwt_verify ( axiam_client_t *  client,
const char *  token,
char **  out_claims_json,
axiam_error_t *  err 
)

Verify a compact JWS (header.payload.signature) using the client's cached JWKS (fetched/refreshed as needed).

On success returns AXIAM_OK and, if out_claims_json is non-NULL, sets *out_claims_json to a heap JSON string of the payload (caller frees with free()). Signature (not exp) is checked.

Failure modes: AXIAM_ERR_AUTH (malformed token, non-EdDSA alg, unknown kid, bad signature), AXIAM_ERR_NETWORK (JWKS could not be fetched).
