1/*
2 * AXIAM C SDK — HTTP transport seam (testability).
3 *
4 * The client performs all HTTP through a function pointer. The default
5 * implementation is libcurl (see axiam_client_config_use_default_transport,
6 * applied automatically). Tests override it with an in-memory fake that
7 * returns canned status/headers/body, exercising the entire logic layer
8 * (request building, JSON, error mapping, CSRF, tenant header, single-flight)
9 * without a live server.
10 */
