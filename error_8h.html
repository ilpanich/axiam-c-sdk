typedef struct axiam_error axiam_error_t

Error detail carrier.

Callers may pass a pointer to a stack-allocated struct to any operation; it is filled on failure. The message buffer is fixed-size and is guaranteed never to contain a raw token (CONTRACT.md §2, §7).
